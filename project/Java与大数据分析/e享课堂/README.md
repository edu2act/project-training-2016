# 2016级项目实训成果展示 

## 《e享课堂》 - Java与大数据分析方向

<img src="./image/logo.png"/>

### 项目简介

e享课堂是一款针对在校大学生的学习平台，给每个人提供“低门槛”当老师的机会，同时可以学习自己感兴趣的课程，平台提供线下预约场地的功能，满足大家组团学习等多样化需求，丰富大家的校园生活。
