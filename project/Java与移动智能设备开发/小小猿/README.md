# 2016级项目实训成果展示 

## 《小小猿》 -  Java与移动智能设备开发方向

###  项目简介

小小猿是一款本土化的少儿编程软件，集练习、教育于一体，在这里既可以通过练习体验编程乐趣、开发编程思维，又可以通过教育获取知识，提升自身编程素养。目前国内少儿编程市场尚未成熟而愈发火爆，本软件着力于为青少年提供一个有效便捷的学习、训练平台。
